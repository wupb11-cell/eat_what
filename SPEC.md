# 饮食推荐助手 - 项目规格说明书（小程序版）

## 1. 项目概述

### 项目名称
EatWhat - 每日饮食推荐助手

### 核心功能
基于营养学原理（低GI），为中国用户推荐每日饮食搭配及详细做法，提供食材采购链接，支持个性化设置和营养分析，通过微信小程序实现每周自动推送。

### 目标用户
- 注重健康饮食的上班族
- 需要控制血糖/体重的人群
- 不擅长烹饪但希望吃得健康的人群

## 2. 技术架构

### 技术栈
- **后端框架**: Python Flask

- **微信集成**: 微信小程序 + 订阅消息

## 3. 功能需求

### 3.1 小程序登录与用户管理
- 微信一键登录（自动获取openid）
- 用户设置存储（永久保存）
- 订阅消息授权管理

### 3.2 菜谱数据库
- 至少50道菜谱，覆盖早中晚三餐
- 每道菜包含：
  - 名称（中餐为主）
  - 分类（早餐/午餐/晚餐/加餐）
  - 食材清单及用量
  - 详细做法步骤
  - 预估成本（元）
  - 烹饪难度（1-5星）
  - GI值（升糖指数）
  - 预估烹饪时间
  - 营养标签（低GI、高蛋白、减肥等）
  - 营养成分（热量、蛋白质、碳水、脂肪、纤维）

### 3.3 推荐算法
- **日期因素**: 根据星期几推荐
- **季节因素**: 春夏推荐清淡，秋冬推荐温热
- **多样性**: 同一周内尽量不重复
- **低GI优先**: 优先选择GI值低的食物
- **成本控制**: 每餐平均成本不超过30元
- **简便性**: 工作日推荐更简单的菜谱
- **个性化**: 根据用户偏好和忌口筛选

### 3.4 个性化设置
- **饮食偏好**:
  - 素食/半素食
  - 低脂/高蛋白
  - 低糖/低碳水
- **忌口设置**:
  - 不吃猪肉/牛肉/海鲜等
  - 过敏食材
- **口味偏好**:
  - 清淡/微辣/重口

### 3.5 营养分析
- **每日营养摄入**:
  - 热量（kcal）
  - 蛋白质（g）
  - 碳水化合物（g）
  - 脂肪（g）
  - 纤维（g）
  - 平均GI值
- **每周营养统计**:
  - 总摄入量
  - 日均摄入
  - 与标准对比
  - 健康建议

### 3.6 微信订阅消息
- **推送时间**: 每周日 18:00 推送下周菜谱
- **推送内容**:
  - 本周7天菜谱概览
  - 每日营养摄入预览
  - 采购清单链接
- **交互功能**:
  - 用户可点击查看今日详细做法
  - 一键跳转采购链接

### 3.7 食材采购链接
- 自动生成美团买菜/京东超市链接
- 按食材分类展示
- 显示每道菜的使用情况

## 4. 数据库结构

### 菜谱表 (recipes)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| name | VARCHAR(100) | 菜名 |
| category | VARCHAR(20) | breakfast/lunch/dinner/snack |
| ingredients | TEXT | JSON格式：["番茄:2个", "鸡蛋:2个"] |
| steps | TEXT | JSON格式：["步骤1", "步骤2"] |
| cost | DECIMAL(6,2) | 预估成本 |
| difficulty | INTEGER | 难度1-5 |
| gi_value | INTEGER | GI值 |
| cook_time | INTEGER | 烹饪时间（分钟） |
| nutrition_tags | TEXT | JSON格式：["低GI", "高蛋白"] |
| season | VARCHAR(20) | 适合季节 |
| calories | INTEGER | 热量（kcal） |
| protein | DECIMAL(5,1) | 蛋白质（g） |
| carbs | DECIMAL(5,1) | 碳水化合物（g） |
| fat | DECIMAL(5,1) | 脂肪（g） |
| fiber | DECIMAL(5,1) | 纤维（g） |

### 用户表 (users)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| openid | VARCHAR(100) | 微信openid（唯一） |
| nickname | VARCHAR(100) | 昵称 |
| avatar_url | VARCHAR(500) | 头像URL |
| settings | TEXT | 用户设置JSON |
| preferences | TEXT | 偏好设置JSON |
| subscribed | INTEGER | 是否订阅消息 |
| created_at | TIMESTAMP | 创建时间 |
| updated_at | TIMESTAMP | 更新时间 |

### 周菜单表 (weekly_menus)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键 |
| start_date | DATE | 开始日期 |
| end_date | DATE | 结束日期 |
| menu_data | TEXT | 菜单JSON数据 |
| created_at | TIMESTAMP | 创建时间 |

## 8. 营养标签说明

### GI值分类
- **低GI**: GI < 55（推荐）
- **中GI**: 55 ≤ GI < 70（适量）
- **高GI**: GI ≥ 70（少食）

### 每日营养标准（成人）
- 热量：1800-2500 kcal
- 蛋白质：50-80 g
- 碳水：200-300 g
- 脂肪：40-60 g
- 纤维：25-35 g

### 营养标签
- 低GI：适合控制血糖
- 高蛋白：适合增肌
- 高纤维：促进消化
- 减脂：适合减肥
- 清淡：少油少盐
